CF1987F2 Interesting Problem (Hard Version) 做题记录

发布于 2025-05-26 | 标签: DP 、 做题记录 | 4分钟 | 665字数

对于一个长 nnn 的序列 aaa，你可以选择一个满足 ai=ia_i=iai​=i 且 i<ni<ni<n 的位置 iii 并删除 aia_iai​ 和 ai+1a_{i+1}ai+1​，删除后两边会拼接。

给定一个长 nnn 的序列 aaa，求最多能进行多少次上述操作。

1≤n≤8001\le n\le 8001≤n≤800，1≤ai≤n1\le a_i\le n1≤ai​≤n。

不会 *2600。

删除的过程不好刻画，考虑某个位置 aia_iai​ 和与它一起被删去的位置 aja_jaj​（i<ji<ji<j），显然 a[i+1,j−1]a_{[i+1,j-1]}a[i+1,j−1]​ 都要被删去，且它们一定先于 (ai,aj)(a_i,a_j)(ai​,aj​) 被删去。

这启发我们设计状态：fl,rf_{l,r}fl,r​ 表示完全删除 a[l,r]a_{[l,r]}a[l,r]​ 至少需要在 [1,l−1][1,l-1][1,l−1] 中执行多少次删除操作。

转移考虑枚举和 ala_lal​ 一起删去的 aka_kak​。显然 l<all<a_ll<al​ 或者 l≡al(mod2)l\not\equiv a_l\pmod 2l​≡al​(mod2) 则 ala_lal​ 就无法被删去，否则设 v=l−al2v=\frac{l-a_l}{2}v=2l−al​​。由于 ala_lal​ 和 aka_kak​ 一起删除，所以 a[l+1,k−1]a_{[l+1,k-1]}a[l+1,k−1]​ 前面最多能执行 vvv 次操作，故要求 fl+1,k−1≤vf_{l+1,k-1}\le vfl+1,k−1​≤v，此时有 max⁡(v,fk+1,r−k−l+12)→fl,r\max(v,f_{k+1,r}-\frac{k-l+1}{2})\to f_{l,r}max(v,fk+1,r​−2k−l+1​)→fl,r​。

边界为 fi+1,i=0f_{i+1,i}=0fi+1,i​=0。

求答案是简单的，设 gig_igi​ 表示 a[1,i]a_{[1,i]}a[1,i]​ 中最多执行多少次操作，利用 fff 转移即可。

时间复杂度 O(n3)O(n^3)O(n3)，代码如下：

#include <iostream>
#include <cstdio>

using namespace std;

const int S=805,inf=1e8;

int n,a[S];
int f[S][S],g[S];

inline void slove()
{
	cin>>n;
	for(int i=1;i<=n;i++) cin>>a[i];
	for(int i=0;i<=n;i++) f[i+1][i]=0;
	for(int len=2;len<=n;len+=2)
		for(int l=1;l<=n-len+1;l++)
		{
			int r=l+len-1;
			f[l][r]=inf;
			if(l>=a[l]&&(l-a[l]&1^1))
			{
				int val=(l-a[l])/2;
				for(int k=l+1;k<=r;k+=2)
					if(f[l+1][k-1]<=val)
					{
						int pre=max(
							val,
							f[k+1][r]-(k-l+1)/2
						);
						f[l][r]=min(f[l][r],pre);
					}
			}
		}
	g[0]=0;
	for(int i=1;i<=n;i++)
	{
		g[i]=g[i-1];
		for(int j=i-1;j>=1;j-=2)
			if(g[j-1]>=f[j][i]) g[i]=max(g[i],g[j-1]+(i-j+1)/2);
	}
	cout<<g[n]<<'\n';
}

int main()
{
	ios::sync_with_stdio(false),cin.tie(0),cout.tie(0);
	int T;
	cin>>T;
	while(T-->0) slove();
	return 0;
}
