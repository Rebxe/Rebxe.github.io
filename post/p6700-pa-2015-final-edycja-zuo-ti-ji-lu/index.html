P6700 [PA 2015 Final] Edycja 做题记录

发布于 2025-12-30 | 标签: 贪心 、 结论 、 做题记录 | 7分钟 | 1266字数

给定两个长度均为 nnn 的字符串 AAA 和 BBB 还有一个正整数 ccc，每次操作可以选择以下一种进行：

  1. 选择一个位置 1≤i≤n1\le i\le n1≤i≤n，将 AiA_iAi​ 修改为任意字符，花费 111 的代价；
  2. 选择两个字符 x,yx,yx,y，将 Ai=xA_i=xAi​=x 的 AiA_iAi​ 全部修改为 yyy，花费 ccc 的代价；

求将 AAA 变为 BBB 的最小代价。

1≤c≤n≤1061\le c\le n\le 10^61≤c≤n≤106，字符集为小写字母。

不难发现，1 类操作一定是在所有 2 类操作之后进行的。

那么有一个初步的想法是每种字母最多进行 111 次 1 类操作，故可以对于每种字母 xxx，求出其最终变为字母 yyy 的代价 Vx,y=c×[x=y]+∑i[Ai=x]V_{x,y}=c\times [x\not=y]+\sum\limits_{i}[A_i\not=x]Vx,y​=c×[x​=y]+i∑​[Ai​​=x]，然后每种字母求出 Vx,∗V_{x,*}Vx,∗​ 的最小值加起来。

可是这样会出现问题。具体的，若 Vx,∗V_{x,*}Vx,∗​ 中最小值是 Vx,yV_{x,y}Vx,y​，那么连接有向边 x→yx\to yx→y。则图肯定是基环树森林，此时每个连通块相对独立，考察每个连通块的形态：

  • 若基环是自环，即一棵树，则可以按照深度从小到大依次操作每个点。此时贪心算的贡献是对的；
  • 若连通块是基环树，即基环大小 ≥2\ge 2≥2 且外面有点 uuu 连到基环上形成类似环上 x→y→zx\to y\to zx→y→z，环外有 u→xu\to xu→x 的情况，则可以先一次操作把 uuu 变到 xxx，这样 uuu 的位置就空出来了。此时贪心算的贡献也是对的；
  • 否则连通块是一个大小 ≥2\ge 2≥2 的环，此时只能先操作完树和基环树的连通块，借用它们的叶子“中转”一次。故不能所有连通块都是环或者自环。此时贪心算的贡献少了一个 ccc；

考虑利用一下这个贪心得到的图 TTT。注意到若最优解对应的图 GGG 中存在一个 TTT 中没有的环 rrr，那么 rrr 中一定有某个点 uuu 满足其选择的出点 vvv 的代价 Vu,vV_{u,v}Vu,v​ 不是 Vu,∗V_{u,*}Vu,∗​ 中最小的。那么将 uuu 的出点改回 TTT 中的那个（假设是 xxx）不仅会减少代价，还会断开这个新环，并且由于给 xxx 连上了一条链，所以也不会新增环。

所以 GGG 中一定不会存在 TTT 中没有的环。

接下来就简单了，状压记录 TTT 中的每个大小 ≥2\ge 2≥2 的环（包括基环树的基环）是否还未断开或形成基环树，转移考虑每次加入的点 iii 的连边情况即可。

最后统计答案需要枚举还剩哪些环需要多一次中转操作，加上额外的代价即可。注意特判没有可供中转的点的情况。

时间复杂度 O(∑22∑2)O(\sum^22^{\frac{\sum}{2}})O(∑222∑​)，其中 ∑\sum∑ 是字符集大小（即 262626），代码如下：

#include <iostream>
#include <cstdio>
#include <vector>

using namespace std;

const int S=1000005,VS=26;
const int inf=1e8;

int n,c;
char a[S],b[S];
int ct[VS][VS],res[VS][VS];
int to[VS],vis[VS];
int tot,idx[VS];
int f[2][1<<(VS/2)];

inline void tmin(int &x,int y)
{
	x=min(x,y);
}

int main()
{
	ios::sync_with_stdio(false),cin.tie(0),cout.tie(0);
	cin>>n>>c;
	cin>>a>>b;
	for(int i=n;i>=1;i--) a[i]=a[i-1]-'a',b[i]=b[i-1]-'a';
	for(int i=1;i<=n;i++) ct[a[i]][b[i]]++;
	for(int i=0;i<VS;i++)
	{
		int all=0;
		for(int j=0;j<VS;j++) all+=ct[i][j];
		for(int j=0;j<VS;j++) res[i][j]=c*(i!=j)+all-ct[i][j];
	}
	for(int i=0;i<VS;i++)
	{
		int v=0;
		for(int j=0;j<VS;j++) if(res[i][j]<res[i][v]) v=j;
		// if(v!=i) cout<<i<<' '<<v<<'\n';
		to[i]=v;
	}
	for(int i=0;i<VS;i++) vis[i]=-1;
	for(int i=0;i<VS;i++)
		if(vis[i]==-1)
		{
			int u=i;
			while(vis[u]==-1) vis[u]=i,u=to[u];
			if(vis[u]==i&&to[u]!=u)
			{
				idx[u]=++tot;
				int x=to[u];
				while(x!=u) idx[x]=tot,x=to[x];
			}
		}
	for(int s=0;s<(1<<tot);s++) f[0][s]=inf;
	f[0][0]=0;
	for(int i=0;i<VS;i++)
	{
		int u=i&1,v=u^1;
		for(int s=0;s<(1<<tot);s++) f[v][s]=inf;
		for(int s=0;s<(1<<tot);s++)
		{
			int x=f[u][s];
			for(int j=0;j<VS;j++)
			{
				int ns;
				if(idx[i]!=0)
				{
					if(j==to[i]) ns=s;
					else
					{
						ns=s|(1<<idx[i]-1);
						if(idx[j]!=0) ns|=(1<<idx[j]-1);
					}
				}
				else
				{
					if(idx[j]==0) ns=s;
					else ns=s|(1<<idx[j]-1);
				}
				tmin(f[v][ns],x+res[i][j]);
			}
		}
	}
	bool flg=true;
	for(int i=0;i<VS;i++) flg&=(idx[i]!=0);
	int ans=inf;
	for(int s=flg;s<(1<<tot);s++)
		tmin(ans,f[VS&1][s]+c*(tot-__builtin_popcount(s)));
	cout<<ans<<'\n';
	return 0;
}
