【zr25年省选联考day9】B. 做菜 做题记录

发布于 2025-04-07 | 标签: DP 、 组合计数 、 做题记录 | 9分钟 | 1660字数

有 nnn 个人，每个人点了一道菜。对于第 iii 个人，做他的菜需要花 aia_iai​ 秒，他吃菜需要花 bib_ibi​ 秒。

你同时只能做一道菜，对于每个人，一旦上菜了他就会马上去吃，吃完后会马上离开。

给定一个整数 kkk，你要求出任意选择其中 kkk 个人且任意安排上菜顺序时，花费的最短时间。其中花费的时间指从做第一道菜开始到最后一个人吃完离开结束的时间差。

现在给定 b[1,n]b_{[1,n]}b[1,n]​ 和 VVV，求 aia_iai​ 在 [1,V][1,V][1,V] 任选的所有 VnV^nVn 种方案中每种方案花费的最少秒数的和。

kkk 也是会改变的，具体的，你要对 k∈[1,n]k\in [1,n]k∈[1,n] 都求出答案。

1≤n≤301\le n\le 301≤n≤30，1≤V≤201\le V\le 201≤V≤20，1≤bi≤6001\le b_i\le 6001≤bi​≤600。

首先对于 x,yx,yx,y 这两个人，两种情况的答案分别是：

max⁡(ax+bx,ax+ay+by)max⁡(ay+by,ay+ax+bx)\max(a_x+b_x,a_x+a_y+b_y)\\ \max(a_y+b_y,a_y+a_x+b_x) max(ax​+bx​,ax​+ay​+by​)max(ay​+by​,ay​+ax​+bx​)

考察 xxx 在 yyy 前面的条件：

max⁡(ax+bx,ax+ay+by)<max⁡(ay+by,ay+ax+bx)\max(a_x+b_x,a_x+a_y+b_y)<\max(a_y+b_y,a_y+a_x+b_x) max(ax​+bx​,ax​+ay​+by​)<max(ay​+by​,ay​+ax​+bx​)

等价于：

{ax+bx<max⁡(ay+by,ay+ax+bx)ax+ay+by<max⁡(ay+by,ay+ax+bx)\begin{cases} a_x+b_x<\max(a_y+b_y,a_y+a_x+b_x)\\a_x+a_y+b_y<\max(a_y+b_y,a_y+a_x+b_x)\end{cases} {ax​+bx​<max(ay​+by​,ay​+ax​+bx​)ax​+ay​+by​<max(ay​+by​,ay​+ax​+bx​)​

第一条显然满足，故等价于：

ax+ay+by<max⁡(ay+by,ay+ax+bx)by<max⁡(by−ax,bx)by<bx\begin{aligned} a_x+a_y+b_y&<\max(a_y+b_y,a_y+a_x+b_x)\\ b_y&<\max(b_y-a_x,b_x)\\ b_y&<b_x \end{aligned} ax​+ay​+by​by​by​​<max(ay​+by​,ay​+ax​+bx​)<max(by​−ax​,bx​)<bx​​

故最优策略是按照 bib_ibi​ 从大到小处理。

考虑 aaa 固定的情况怎么算答案，按 bbb 从小到大排序（倒着做），设 fi,jf_{i,j}fi,j​ 表示前 iii 个人选了 jjj 个人的答案，那么有：

fi,j=min⁡(fi−1,j,max⁡(fi−1,j−1,bi)+ai)f_{i,j}=\min(f_{i-1,j},\max(f_{i-1,j-1},b_i)+a_i) fi,j​=min(fi−1,j​,max(fi−1,j−1​,bi​)+ai​)

注意到由于 bib_ibi​ 不降，故若 fx,j≤bif_{x,j}\le b_ifx,j​≤bi​ 则有 f[x+1,n],j=fx,jf_{[x+1,n],j}=f_{x,j}f[x+1,n],j​=fx,j​，即 f∗,jf_{*,j}f∗,j​ 不会再更新。这启发我们只维护 fi,[l,n]f_{i,[l,n]}fi,[l,n]​，其中 lll 是满足 fi,l>bif_{i,l}>b_ifi,l​>bi​ 的最小的 lll。

为 fi,[l,n]f_{i,[l,n]}fi,[l,n]​ 定义差分数组 ci,∗c_{i,*}ci,∗​ 如下：

  • 若 j=lj=lj=l 则 ci,j=fi,j−bic_{i,j}=f_{i,j}-b_ici,j​=fi,j​−bi​；
  • 否则 ci,j=fi,j−fi,j−1c_{i,j}=f_{i,j}-f_{i,j-1}ci,j​=fi,j​−fi,j−1​；

那么每次转移相当于在 ci,∗c_{i,*}ci,∗​ 某处插入 aia_iai​，则可以归纳证明 ci,∗c_{i,*}ci,∗​ 是单调递增的。

那么每次转移形如：

  • 设 d=bi−bi−1d=b_i-b_{i-1}d=bi​−bi−1​，则不断用 ddd 去消除 ccc 的开头元素（两个一起减一，若开头元素减没了就 pop 掉），若 ccc 消完了则 ddd 和虚空消，直到 ddd 消完；
  • 找到 ccc 中第一个大于 aia_iai​ 的位置，在其前面插入 aia_iai​；

最后答案就是 ccc 前 kkk 次 pop 期间消除的总次数。

计数考虑拆每个 aia_iai​ 的贡献，那么要记录 c 中 ≤ai\le a_i≤ai​ 的数的和 smsmsm、aia_iai​ 的值 xxx 以及在 aia_iai​ 前面 pop 的数的个数 kkk。

对于 [i+1,n][i+1,n][i+1,n] 这些元素，转移是简单的；而对于 [1,i][1,i][1,i] 这些元素，考虑在它们加入的时候就决定是在 iii 之前还是在 iii 之后被 pop 的，那么就需要记录 ≤ai\le a_i≤ai​ 的数的最大值 lblblb、≤ai\le a_i≤ai​ 的数的和 smlsmlsml 以及 >ai>a_i>ai​ 的数的最小值 rbrbrb，转移过程中需保证 rb>1rb>1rb>1。

时间复杂度 O(n3V4)O(n^3V^4)O(n3V4)，可以进一步优化但是没必要，严重跑不满。代码如下：

#include <iostream>
#include <algorithm>
#include <cstdio>
#include <cstring>

using namespace std;

const int S=35,VS=25,MS=605;

#define p 998244353

int n,m,q;
int b[S],pw[S];
int f[2][VS][MS][VS][S]; // i,lb,sml,rb,ctl
int g[2][MS][VS][S]; // i,sum,x,k
int ans[S];

inline void add(int &x,int y)
{
	x+=y;
	if(x>=p) x-=p;
}

int main()
{
	// ios::sync_with_stdio(false),cin.tie(0),cout.tie(0);
	cin>>n>>m;
	for(int i=1;i<=n;i++) cin>>b[i];
	pw[0]=1;
	for(int i=1;i<=n;i++) pw[i]=1ll*pw[i-1]*m%p;
	sort(b+1,b+n+1);
	f[0][0][0][m+1][0]=1;
	for(int i=1;i<=n;i++)
	{
		int u=i&1,v=u^1;
		memset(f[u],0,sizeof(f[u]));
		memset(g[u],0,sizeof(g[u]));
		// f
		for(int lb=0;lb<=m;lb++)
		for(int sml=lb;sml<=(i-1)*m;sml++)
		for(int rb=lb;rb<=m+1;rb++)
		for(int k=0;k<=i-1;k++)
		{
			int x=f[v][lb][sml][rb][k];
			if(x==0) continue;
			int d=b[i]-b[i-1];
			int nlb=lb,nsml=sml,nrb=rb;
			int de=min(nsml,d);
			nsml-=de,d-=de;
			if(nsml<=nlb) nlb=nsml;
			if(d!=0&&nrb==m+1) // empty
				add(ans[k+1],1ll*x*d%p*pw[n-i+1]%p);
			else
			{
				nrb-=d;
				if(nrb<max(nlb,1)) continue;
			}
			// f->f
			for(int a=1;a<=m;a++)
			{
				// <=x
				if(a<=nrb)
					add(f[u][max(nlb,a)][nsml+a][nrb][k+1],x);
				// >x
				if(a>nlb)
					add(f[u][nlb][nsml][min(nrb,a)][k],x);
			}
			// f->g
			for(int a=max(nlb,1);a<nrb;a++)
				add(g[u][nsml+a][a][k+1],x);
		}
		// g->g
		for(int sm=1;sm<=(i-1)*m;sm++)
		for(int a=1;a<=m&&a<=sm;a++)
		for(int k=1;k<=i-1;k++)
		{
			int x=g[v][sm][a][k];
			if(x==0) continue;
			int d=b[i]-b[i-1];
			if(sm<=d) add(ans[k],1ll*x*a%p*pw[n-i+1]%p);
			else
			{
				int nsm=sm-d,na=a;
				if(nsm<a)
				{
					na=nsm;
					add(ans[k],1ll*x*(a-na)%p*pw[n-i+1]%p);
				}
				for(int b=1;b<=m;b++)
				{
					if(b<na) // <x
						add(g[u][nsm+b][na][k+1],x);
					else // >=x
						add(g[u][nsm][na][k],x);
				}
			}
		}
	}
	for(int sm=1;sm<=n*m;sm++)
	for(int a=1;a<=m&&a<=sm;a++)
	for(int k=1;k<=n;k++)
	{
		int x=g[n&1][sm][a][k];
		if(x==0) continue;
		add(ans[k],1ll*x*a%p);
	}
	for(int i=1;i<=n;i++) add(ans[i],ans[i-1]);
	cin>>q;
	while(q-->0)
	{
		int x;
		cin>>x;
		cout<<ans[x]<<'\n';
	}
	return 0;
}
